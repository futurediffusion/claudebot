# Banana Pro Director Protocol

Este protocolo transforma al agente en un Director Creativo capaz de orquestar escenas complejas en Nano Banana Pro con precision de set de fotografia real.

## El Framework SLCT (Optimizado)

### S - SUBJECT (Sujeto)
- No solo describas: **Dirige**.
- Describe micro-detalles: poros, vello facial sutil, reflejos en los ojos (catchlights), textura de la tela.
- **Accion**: Describe el peso corporal (ej: "leaning naturally", "shoulders hunched").

### L - LIGHTING (Iluminacion)
- Define la fisica: "Rim lighting", "Chiaroscuro", "Volumetric window light".
- Define el rebote: "Reflected light from the polished floor".

### C - CAMERA (Camara y Composicion)
- **Lentes**: 85mm f/1.8 (Retrato), 35mm (Lifestyle), 24mm (Gran angular distorsionado).
- **Angulos**: "Extreme low angle", "Top-down flat lay", "Dutch angle for tension".

### T - TEXT & DETAILS (Textura y Texto)
- Especifica materiales: "Matte cotton", "Glossy latex", "Distressed leather".

## Modos Especiales

### 1. Character Consistency (Identity Lock)
Prompt Base: "Maintain the exact facial structure, skin tone, and body proportions from Image [ID]. Change only the environment to [Lugar]. Preserve the original camera angle."

### 2. Virtual Try-On (Scene Merge)
Prompt Base: "Act as a professional virtual stylist. Dress the model from Image [ID] in the garment from Image [ID]. Ensure the fabric drapes naturally over the pose. Preserve shadows and lighting coherence."

### 3. Structure Reference (Geometry Lock)
Prompt Base: "Use the room layout and geometry from Image [ID] as the base structure. Apply the interior design style and color palette from Image [ID]. Keep all walls and proportions unchanged."
