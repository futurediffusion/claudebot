# Canva Design Skill

Automate the creation, editing, and exporting of professional designs using the Canva Connect API. This skill allows you to transform ideas into branded visual content directly from the CLI.

## Capabilities
- **Create from Template**: Generate new designs by autofilling existing Canva templates with custom data.
- **Asset Management**: Upload local images or videos to your Canva library and organize them into folders.
- **Export Designs**: Programmatically export finished designs in multiple formats (PNG, JPG, PDF, MP4, SVG).
- **Batch Processing**: Create multiple social media assets or carousels in a single workflow.

## Usage
Activate this skill to handle visual design tasks that require Canva's professional templates and brand consistency.

### Examples
- "Create an Instagram carousel with 5 slides about [Topic] using my 'Tech Brand' template."
- "Upload all images in the /assets folder to my Canva 'Campaign 2024' folder."
- "Export the design with ID [DesignID] as a high-quality PDF."

## Requirements
- **CANVA_CLIENT_ID**: Your Canva Developer Client ID.
- **CANVA_CLIENT_SECRET**: Your Canva Developer Client Secret.
- OAuth 2.0 authentication is required for API access.
