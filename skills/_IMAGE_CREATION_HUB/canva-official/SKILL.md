# Official Canva MCP Skill

Directly interface with the official Canva Model Context Protocol (MCP) server to search, create, and edit designs within the Claude conversation.

## Capabilities
- **Design Creation**: Generate new Canva designs from scratch or based on existing templates.
- **Visual Search**: Search your Canva library and discover new elements or templates.
- **Smart Editing**: Modify layouts, resize visuals, and adjust elements using natural language.
- **Brand Kit Integration**: Automatically apply your Canva Brand Kit (colors, fonts, logos) to AI-generated drafts.
- **Interactive Previews**: Render UI widgets and previews of designs directly in the chat.

## Usage
Use this skill for professional-grade design tasks that require the full power of Canva's ecosystem combined with Claude's reasoning.

### Examples
- "Search for a minimalist Instagram template and create a new design for my brand."
- "Refine this design: make the background darker and the text bolder."
- "Resize this square post for a LinkedIn header and an Instagram Story."

## Connection
- **Endpoint**: `https://mcp.canva.com/mcp`
- **Transport**: HTTP/SSE
- **Official URL**: [Canva Developer Portal](https://www.canva.com/developers/)
