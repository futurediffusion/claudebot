# Web Design Hub

8 skills for frontend design, animations, web artifacts, and testing.

## Skills

| Skill | Purpose |
|-------|---------|
| algorithmic-art | Generative algorithmic art with code: fractals, cellular automata, L-systems |
| canvas-design | HTML Canvas and WebGL creative coding |
| frontend-design | Design systems, component libraries, responsive layouts |
| gsap-skills | GSAP animation library: timelines, ScrollTrigger, morphing |
| mcp-builder | Build and deploy MCP (Model Context Protocol) servers |
| theme-factory | CSS theme generation: tokens, dark/light mode, brand themes |
| web-artifacts-builder | Build complete web artifacts: landing pages, dashboards, tools |
| webapp-testing | E2E and integration testing for web applications |

## Usage

To load a specific skill, read its SKILL.md:
```
/read _WEB_DESIGN_HUB/frontend-design/SKILL.md
/read _WEB_DESIGN_HUB/gsap-skills/SKILL.md
/read _WEB_DESIGN_HUB/web-artifacts-builder/SKILL.md
```
