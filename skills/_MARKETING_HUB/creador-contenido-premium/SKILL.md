# Skill: Creador de Contenido Premium

Este skill transforma conceptos crudos en narrativas visuales de alta gama para redes sociales (Instagram, LinkedIn, Twitter). Combina la estrategia de investigación con una dirección de arte editorial de "clase mundial".

## Directrices de Diseño (Estándar High-End)
- **Cero "AI Slop"**: Prohibido usar fuentes genéricas como Arial o Inter. Usar tipografías con carácter (Serif para elegancia, Sans-serif ultra-fina para modernidad).
- **Estética Editorial**: El contenido debe parecer una revista de lujo. Uso generoso del espacio negativo y composiciones asimétricas.
- **Paletas Curadas**: Nada de degradados púrpuras clichés. Usar tonos tierra, minerales, negros profundos y acentos dorados o cobre.
- **Textura Física**: Aplicar siempre una capa de grano fino para dar profundidad y una sensación táctil a la imagen.

## Flujo de Trabajo
1. **Investigación (Insight)**: Identificar una verdad profunda del usuario antes de diseñar.
2. **Filosofía Visual**: Definir el "sentimiento" de la colección.
3. **Ejecución Maestra**: Generar imágenes con tipografía perfecta y layout de museo.

## Comandos Rápidos
- `/carrusel-lujo [tema]`: Genera un carrusel de 5 slides con estética editorial.
- `/post-unico [tema]`: Crea una sola pieza de arte visual con un mensaje potente.
